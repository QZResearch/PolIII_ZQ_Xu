# PolIII_ZQ_Xu

``Structural basis of high-processivity high-fidelity DNA replication by Escherichia coli DNA polymerase III'' Xu et al. DOI: to be updated

## structures

Structures resolved from experiment

## w_dGTP

Files for simulating system containing dGTP, including 

- `tleap` file to generate the whole system;
- the parameters files `dGTP` for dGTP and `ff-DNA-OL24` for DNA;
- four directories for `minimization`, `heating`, `equilibrating at NPT ensemble`, and `production`

## wo_dGTP

Files for simulating system without dGTP, including 

- `tleap` file to generate the whole system;
- the parameters files `ff-DNA-OL24` for DNA;
- four directories for `minimization`, `heating`, `equilibrating at NPT ensemble`, and `production`

## scripts

script for analysing distance formed between two residues
